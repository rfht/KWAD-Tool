###KWAD Tool

This is a simple tool for listing/extracting files packed with Klei's `.kwad` format.

###Usage

You have to have **Python 2.7** installed in order to use this tool.

```bash
python kwad-tool.py PATH_TO_KWAD_FILE [EXTRACTING_PATH]
```
